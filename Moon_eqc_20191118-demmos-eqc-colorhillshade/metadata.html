Description:
Abstract:
This is a color hillshade visualization of an elevation mosaic produced by combining stereo pairs generated from LROC NAC images, for the southwestern area of Lacus Mortis, including Rimae Bürg and the Lacus Mortis Pit. Each stereo pair was individually aligned to LOLA laser altimetry.
Purpose:
This color hillshade visualization of provides a high resolution representation of the southwestern area of Lacus Mortis.

Process_Description:
The processing approach to generate and mosaic the DEMs uses the Solar System Treks Lunar Mapping and Modelling Pipeline, which uses Ames Stereo Pipeline for stereo reconstruction. Source images were: M1103380406L, M1103380406R, M1103401840L, M1103401840R, M1105737674L, M1105737674R, M1105759104L, M1105759104R, M1134024492L, M1192879954L, M1192879954R, M1192901040L, M1192901040R, M1208214370R.
The color hillshade visualization of the dem was completed using a combination of Geospatial Data Abstraction Library (GDAL) and Ames Stereo Pipeline. To derive this colorized shaded-relief these three GDAL commands were used: (1) gdaldem hillshade input_dem output_hillshade -z 1 (ZFactor) -az 315 (Azimuth) -alt 45 (Altitude) (2) gdaldem color-relief input_dem color_text_file output_color_relief_map (3) hsv_merge output_color_relief_map output_hillshade output_color-hillshade. (from Ames Stereo Pipeline)
Pixels do not contain any of the original explicit elevation information. The colorized shaded-relief represents approximate elevations based on the original topographic data. Elevations values are mapped based on the defined red, green, and blue mapping listed below. All NoData values are mapped to 0,0,0 (black). Thus 0 percent (purple) maps to minimum elevation and 100 percent (white) maps to maximum using a linear stretch between percentages.
nv 0 0 0 //No Data; -2698 255 120 255 //purple; -2635 120 120 255 //blue; -2571 120 255 255 //aqua; -2508 120 255 120 //green; -2445 255 255 120 //yellow; -2382 255 120 120 //red; -2066 255 255 255 //white
