Description:
Abstract:
This is an orthoimage mosaic produced by combining LROC NAC images, for the southwestern area of Lacus Mortis, including Rimae Bürg and the Lacus Mortis Pit. The images are projected onto a DEM mosaic produced by stereo reconstructions in which each of the images is paired with another LROC NAC.

Data_Quality_Information:
Logical_Consistency_Report:
These images were projected onto DEMs from stereo pair point clouds. Most were then individually aligned to LOLA, except for pairs M1108110075RExxM1138737954RE and M1108110075LExxM1123434946RE for which stereo reconstruction quality is poor. These pairs were added in the interest of fully covering the notional landing ellipse from the Astrobotic Peregrine Payload User's Guide with imagery.
Completeness_Report:
There are some nodata pixels in this orthoimage which reflect nodata values in the DEM onto which it was projected, mostly in shadowed craters.

Process_Description:
The processing approach begain with generating DEMs uses the Solar System Treks Lunar Mapping and Modelling Pipeline, which uses AMES Stereo Pipeline for stereo reconstruction. Source images were: M1103380406L, M1103380406R, M1103401840L, M1103401840R, M1105737674L, M1105737674R, M1105759104L, M1105759104R, M1134024492L, M1192879954L, M1192879954R, M1192901040L, M1192901040R, M1208214370R, M1108110075L, M1123434946R, M1108110075R, and M1138737954R. Orthoimages for the left image of each pair were produced using "point2dem --orthoimage" . The individual images were then merged.
