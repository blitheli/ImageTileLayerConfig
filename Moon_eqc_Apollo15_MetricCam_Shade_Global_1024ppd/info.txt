Abstract :The Apollo 15 DEM hillshade is generated using the hillshade
                            function in the Vision Workbench package. The DEM hillshade values are
                            stored as 8-bit signed integer at 14.8m/pixel resolution. These DEM
                            hillshade files were generated using the Vision Workbench library: a
                            software tool produced by the NASA Ames Intelligent Robotics Group
                            (IRG)for automated processing of stereo imagery.
