Apollo 16 Metric Cam DEM, Confidence
Layer ID:Moon_eqc_Apollo16_MetricCam_Conf_Global_1024ppd
bbox:[-179.9995117, -21.9995117, 179.9995117, 18.0004883]
WMTS Endpoint :https://trek.nasa.gov/tiles/Moon/EQ/Apollo16_MetricCam_Conf_Global_1024ppd
Abstract :The Apollo 16 DEM confidence data is generated using the standard deviation of all the available Apollo 16 DEM files. The DEM confidence values are stored as 8-bit signed integer at 14.8m/pixel resolution. The values in the confidence file are derived from the precision values. The value range from 10 to 14 represents poor confidence (10) to very good confidence (14), with 15 = manually edited. The confidence values in the range 10-14 represent values of the standard deviation stored in the precision files as follows: value 10 for standard deviation larger than 100m, value 11 for standard deviation in the range 75-100m, value 12 for standard deviation in the range 50-75m, value 13 for standard deviation in the range 25-50m, and value 14 for standard deviation in the range 0-25m. These DEM confidence files were generated using the Ames Stereo Pipeline: a software tool produced by the NASA Ames Intelligent Robotics Group (IRG) for automated processing of stereo imagery.
