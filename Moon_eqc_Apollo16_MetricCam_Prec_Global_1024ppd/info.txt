Apollo 16 Metric Cam DEM, Precision
Layer ID:Moon_eqc_Apollo16_MetricCam_Prec_Global_1024ppd
bbox:[-179.9995117, -21.9995117, 179.9995117, 18.0004883]
WMTS Endpoint :https://trek.nasa.gov/tiles/Moon/EQ/Apollo16_MetricCam_Prec_Global_1024ppd
Abstract :The Apollo 16 DEM precision is generated using the standard deviation of all the available Apollo 16 DEM files. The DEM precision units are stored as 16-bit signed integer at 14.8m/pixel resolution. The DEM units are represented in meters. These DEM precision files were generated using the Ames Stereo Pipeline: a software tool produced by the NASA Ames Intelligent Robotics Group (IRG) for automated processing of stereo imagery.
阿波罗16号数字高程模型精度是使用所有可用的阿波罗16号DEM文件的标准偏差生成的。DEM精度单位存储为16位带符号整数，分辨率为14.8m/像素。DEM单位以米为单位。这些DEM精度文件是使用艾姆斯立体管道生成的：这是美国国家航空航天局艾姆斯智能机器人小组（IRG）生产的一种用于立体图像自动处理的软件工具。
