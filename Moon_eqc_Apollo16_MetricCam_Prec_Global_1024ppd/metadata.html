Description:
Abstract:
The Apollo 16 DEM precision is generated using the standard deviation of all the available Apollo 16 DEM files. The DEM precision units are stored as 16-bit signed integer at 14.8m/pixel resolution. The DEM units are represented in meters. These DEM precision files were generated using the Ames Stereo Pipeline: a software tool produced by the NASA Ames Intelligent Robotics Group (IRG) for automated processing of stereo imagery.
Purpose:
The Apollo 16 DEM precision data provides an estimated error value for each pixel which incorporates both the stereo and bundle adjustment errors.

The Apollo 16 DEM precison maps are generated using the elevation expected data standard deviation of all the available Apollo 16 DEM files. The precision values of the A16 DEM zone are captured in the file, Apollo16_MetricCam_Prec_Global_1024ppd.tif.
Completeness_Report:
The accuracy and precision of digital elevation map is lower in areas where the images where corrupted by noise in the camera path or scanning artifacts. While the effect of image noise is reduced by the our robust stereo matching algorithm some of the artifacts remain present in the final product. Other areas of low precision and accuracy include shadow regions where stereo correlation has poor quality and areas along the image boundaries.

Process_Step:
Process_Description:
The Apollo 16 DEM is computed from stereo pairs (adjacent images within each orbit) of the Apollo 16 mission. The process uses the bundle adjustment solution and Bayesian subpixel correlator in Ames Stereo Pipeline package to reduce the uncertainties in camera position and reduce the effect of image noise. Elevation values extracted from overlapping stereo pairs (each point on the Lunar surface is seen by more than two images) are averaged and stored as the best elevation estimate. The standard deviation of all elevation values is used as a measure of precision. The range of Apollo images used here include all nadir looking images of the Apollo 16 Metric zone: AS16-M-0003.cub - AS16-M-0173.cub, AS16-M-0313.cub - AS16-M-0451.cub, AS16-M-0853.cub - AS16-M-0998.cub, AS16-M-1145.cub - AS16-M-1288.cub, AS16-M-1552.cub - AS16-M-1689.cub, AS16-M-1841.cub - AS16-M-1985.cub, AS16-M-2069.cub - AS16-M-2216.cub, AS16-M-2693.cub - AS16-M-2999.cub.
