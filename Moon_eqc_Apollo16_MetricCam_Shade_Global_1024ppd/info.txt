Abstract : The Apollo 16 DEM shade is generated using the hillshade function in Vision Workbench package. The DEM shade values are stored as 8-bit signed integer at 14.8m/pixel resolution. These DEM shade files were generated using the Vision Workbench library: a software tool produced by the NASA Ames Intelligent Robotics Group (IRG) for automated processing of stereo imagery.
