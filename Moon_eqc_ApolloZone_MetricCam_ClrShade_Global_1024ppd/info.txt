Abstract :The Apollo Zone DEM Color Hillshade is generated using the colormap and hillshade functions in Vision Workbench for the DEM files. The min and max values of the colormap (legend.png) correspond to the range [-8499, 10208] in meters relative to the average moon radius of 1737.4km. The DEM color hillshade values are stored as 8-bit signed integer at 30m/pixel resolution. These DEM color hillshade files were generated using the Vision Workbench library: a software tool produced by the NASA Ames Intelligent Robotics Group (IRG) for automated processing of stereo imagery.
