Description:
Abstract:
This is a colorized confidence map associated with the Digital Elevation Model (DEM) extracted from the Lunar Reconnaissance Orbiter Camera (LROC) Narrow Angle Camera (NAC) geometric stereo images from the Lunar Reconnaissance Orbiter mission. The original data product is a DEM from stereo images acquired at approximately 0.5 m/p resolution, and the DEM is set at 2m/p, downsampled to minimize noise. This DEM covers the Marius Hills site. The DEM is controlled to the Lunar Orbiter Laser Altimeter (LOLA) dataset. DEM values referred to a radius of 1737400 meters. In the confidence data, the values are derived from precision values. For the colorized confidence, the value range from 10 to 14 was used, ranging from poor confidence (10) to very good confidence (14). This range was mapped to a color map varying from yellow to green. Manually interpolated values (15) are mapped to blue. All other values are mapped to fully transparent pixels.
Purpose:

Data_Quality_Information:
Logical_Consistency_Report:
For each DEM pixel, a derived confidence value is also assigned in the file named LRO_NAC_Conf_14N304E_2mp.tif. The confidence values used to assess the DEM pixels are: 0 = NoDATA, outside boundary; 1 = Shadowed; 2 = Saturated; 3 = Suspicious (edge, corner, did not correlate, etc); 4 = Interpolated/Extrapolated from neighbor pixels; 10-14 = Value range of successful correlations; 15 = Manually interpolated.
Completeness_Report:
Where the images contained stereo overlap, elevation values were extracted for the DEM except for when one of the images was in shadow.
Positional_Accuracy:
Horizontal_Positional_Accuracy:
Horizontal_Positional_Accuracy_Report:
To compute the absolute horizontal accuracy, three of the sixteen LOLA tracks crossing the DEM were selected at random for comparison. The data set that the LOLA tracks were chosen from contains tracks that were not crossover corrected. Multiple tracks were sampled to account for this and ensure that offsets were consistent. The DEM was registered to these tracks to calculate the horizontal offset relative to LOLA and therefore find the absolute horizontal offset. The average horizontal offset, relative to LOLA data, is 30.5 meters. The maximum horizontal offset, relative to LOLA data, was 48.8 meters. Since the accuracy of the LOLA data is 20 meters, the maximum absolute horizontal offset for this site is 68.8 meters.
Quantitative_Horizontal_Positional_Accuracy_Assessment:
Horizontal_Positional_Accuracy_Value: 68.8
Horizontal_Positional_Accuracy_Explanation:
The relative horizontal accuracy (pixel to pixel across the DEM) is the same as the spatial resolution of the DEM which is 2m. The absolute horizontal accuracy is dependent upon the absolute horizontal accuracy of the LOLA data. When this DEM was made, the absolute horizontal accuracy of the LOLA data was 20 meters.
Vertical_Positional_Accuracy:
Vertical_Positional_Accuracy_Report:
To compute the absolute horizontal accuracy three of the sixteen LOLA tracks crossing the DEM were selected at random for comparison. The data set that the LOLA tracks were chosen from contains tracks that were not crossover corrected. Multiple tracks were sampled to account for this and ensure that offsets were consistent. The DEM was registered to these tracks to calculate the elevation offset to LOLA and therefore find the absolute vertical error. The average elevation offset, relative to the LOLA data, is 1.12 meters. The maximum elevation offset, relative to the LOLA offset was 3.20 meters. Since the current elevation accuracy for LOLA data is 1 meter, the absolute elevation offset for this site is 4.20 meters.
Quantitative_Vertical_Positional_Accuracy_Assessment:
Vertical_Positional_Accuracy_Value: 4.20
Vertical_Positional_Accuracy_Explanation:
Since the DEM is tied to LOLA, the vertical accuracy is limited by both the absolute vertical accuracy of the LOLA data and the Expected Vertical Precision (the relative precision) of the DEM. The absolute vertical accuracy of the LOLA data is 1 meter. The relative precision of the DEM is expected to be 1 meter, but can be as large as 2 meters.
