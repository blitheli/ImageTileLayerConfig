Description:
Abstract:
This is a colorized confidence map associated with the Digital Elevation Model (DEM) extracted from the Lunar Reconnaissance Orbiter Camera (LROC) Narrow Angle Camera (NAC) geometric stereo images from the Lunar Reconnaissance Orbiter mission. The original data product is a DEM from stereo images acquired at approximately 0.5 m/p resolution, and the DEM is set at 2m/p, downsampled to minimize noise. This DEM covers the Mare Moscoviense site. The DEM is controlled to the Lunar Orbiter Laser Altimeter (LOLA) dataset. DEM values referred to a radius of 1737400 meters. In the confidence data, the values are derived from precision values. For the colorized confidence, the value range from 10 to 14 was used, ranging from poor confidence (10) to very good confidence (14). This range was mapped to a color map varying from yellow to green. Manually interpolated values (15) are mapped to blue. All other values are mapped to fully transparent pixels.
Purpose:
This confidence color image provides a visual assessment of the goodness of the data in the Mare Moscoviense DEM.

Data_Quality_Information:
Logical_Consistency_Report:
For each DEM pixel, a derived confidence value is also assigned in the file named LRO_NAC_Conf_26N150E_200cmp.tif. The confidence values used to assess the DEM pixels are: 0 = NoDATA, outside boundary; 1 = Shadowed; 2 = Saturated; 3 = Suspicious (edge, corner, did not correlate, etc); 4 = Interpolated/Extrapolated from neighbor pixels; 10-14 = Value range of successful correlations; 15 = Manually interpolated.
Completeness_Report:
Where the images contained stereo overlap, elevation values were extracted for the DEM except for when one of the images was in shadow.

Horizontal_Positional_Accuracy:
Horizontal_Positional_Accuracy_Report:
The horizontal accuracy measured is the RMS in the X (longitude) and Y (latitude) directions in meters of the final triangulation solution as reported in SOCET Set. The RMS values are X = 21.84 and Y = 70.88. The overall RMS of the solution is in pixels. The pixel scale of the input images is 0.5 m. Overall RMS = 0.8049. The horizontal precision of the DEM is the same as the post spacing, which is 2 m. The images used in this project were controlled to LOLA tracks that were determined to be self-consistent, but which may not have been included in the crossover solution. The absolute accuracy of the crossover corrected LOLA data is 20 meters.
